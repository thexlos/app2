# MVP Scope

## Included in this foundation

- Multi-business mock workspaces and switcher
- Mobile-first navigation and responsive shell
- Home, Customers, Money, Create, Help first versions
- Setup checklist and Business Kit preview/apply
- Customer detail and related records
- Estimates, invoices, progress invoices, change orders
- Accepted estimate protection, version history, customer view, note-required responses
- Workshop starter surfaces, File Vault structure, Help request, Helper Access concept
- Task-specific Create builder/guided/help routing and a support-only Help tab
- Builder, email, client approval, official document, and generated PDF delivery surfaces
- Provider placeholder screens/services
- Typed mock entities, editable service catalog, docs, schema plan, and tests

## Explicitly not included

- Authentication, database persistence, file upload/storage, malware scanning
- Live customer messages, emails, SMS, payments, posting, accounting sync, PDF generation
- Production signed links, real signatures, IP/device capture
- Admin pricing editor, full template editor, reports, scheduling, dark mode

Mock actions reset on reload and never claim external completion.
