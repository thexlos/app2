# Roadmap

## Phase 1 — Trusted backend foundation

Authentication, business membership/roles, PostgreSQL persistence, row-level tenant enforcement, immutable versions, audit logs, signed public links, file storage/scanning, and automated authorization tests.

## Phase 2 — Core operations

Real customer/lead CRUD, estimate editor, PDF snapshots, account-free approvals, invoices, payment recording, progress billing, change orders, reusable templates, and notification preferences.

## Phase 3 — Workshop and service delivery

QR generation/testing, flyer/card exports, print checker, lead forms/public pages, file vault/project folders, Helper Access, service quotes, approvals, and admin-managed service catalog.

## Phase 4 — Integrations

QuickBooks read import first, then controlled final-record sync; social account connections and publishing; supported Canva collaboration; email/SMS/payment providers.

## Phase 5 — Desktop depth

Reporting, bulk customer management, sync conflict review, template builder, export center, advanced visibility controls, and customizable dashboard settings.
